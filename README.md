# gms-vue

> 基于SpringBoot + Vue开发的体育馆管理系统前端
>
> ### 项目链接：[体育馆管理系统](http://gms.atiantts.xyz/)
>
> ### **后端部分请参阅[gms后端](https://github.com/OrionAAAA/GMS-springboot)**



### 项目下载后如何运行？

```
1. npm install

2. npm run serve
```
